## [Unreleased]

### Added

- CLI with `init`, `serve`, `create`, `list`, `switch`, `delete`, `status`, `reset`, and `snapshot` commands
- Interactive TUI prompts and progress indicators
- Configuration management with file and environment variable support
- Multi-format output (table, JSON, YAML)
- Multi-platform build support (Linux, macOS, Windows; amd64, arm64)
- Docker image publishing to Docker Hub and GitHub Container Registry
- Homebrew formula for macOS/Linux installation
- CI pipeline with linting, testing, security scanning, and multi-platform builds
